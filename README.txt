# HOTEL MANAGEMENT SYSTEM - Setup Guide

## Project Structure
This Hotel Management System consists of multiple departments, each with its own functionality.


**Department Files:**

Hiring Department:
- HiringDepartment.java
- Applicant.java
- Employee.java

Service Department:
- ServiceDepartment.java
- ServiceRequest.java
- ServiceStaff.java

Front Office:
- FrontOffice.java
- Room.java
- Guest.java

Culinary Department:
- CulinaryDepartment.java
- MenuItem.java
- Order.java

Logistics Department:
- LogisticsDepartment.java
- Resource.java
- Delivery.java

Air Company Department:
- AirCompanyDepartment.java
- Flight.java
- Ticket.java

Placeholder Departments:
- SalesMarketingDepartment.java
- RecreationalFacilities.java


## Departments Overview

1. **Hiring Department** - Manage job applicants and employees
2. **Service Department** - Handle customer service requests
3. **Front Office** - Room booking, check-in, and check-out
4. **Culinary Department** - Manage menu items and food orders
5. **Logistics Department** - Inventory and delivery management
6. **Air Company Collaboration** - Book flight tickets for guests
7. **Sales/Marketing** - (Under construction)
8. **Recreational Facilities** - (Under construction)
